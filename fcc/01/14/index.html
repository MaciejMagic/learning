Introduction

Python is a great introductory programming language and can be used for wide array of scenarios, like:

  • General Scripting
  • Website - database integration
  • Backend of Webapps
  • Artificial Intelligence
  • Machine Learning

Main programming is defining functions. Here's a very short and simple one:

print("Hello, world!")

One can define variables like in any other language.

Variables

Unlike in C, you can define a variable with just its name, like so:

x = 12

You do not have to define a data type, but it's sometimes better to force it. You can do so by encasing the definition in a function, like so:

x = int(12)
Functions

Functions in Python do not use curly braces. Indentation is used instead.

def addition(x, y): return x + y

Above is a function that takes two arguments and adds them together with a simple math equation and then returns the value.

Statements

One can write many different compound statements in Python to create logic for programs. Statements known from other languagese, like:

  • if
  • for
  • while
  • try
  • with

For example:

for i in range(5): print(i)
